# La Ruta de un LLM: De las Matemáticas al Despliegue

¡Hola! Este repositorio es una guía paso a paso creada para entender el ciclo de vida de un Large Language Model (LLM), pensada especialmente para estudiantes de Matemáticas Aplicadas y Computación, y entusiastas de los datos.

## Objetivo
Desmitificar la caja negra de los LLMs. Aquí no solo consumimos APIs, sino que explicamos las matemáticas, la arquitectura y el código necesario para entrenar, afinar y desplegar estos modelos.

## Mapa de Ruta (Roadmap)
1. **[Fundamentos](link-a-carpeta)**: Vectores, embeddings y NLP clásico.
2. **[El Transformer](link-a-carpeta)**: Mecanismos de atención.
3. **[Fine-Tuning](link-a-carpeta)**: Cómo adaptar un LLM a tus propios datos.
4. **[RAG](link-a-carpeta)**: Dándole memoria al modelo.
5. **[Despliegue Full Stack](link-a-carpeta)**: Llevando el modelo a producción web.

## Tecnologías utilizadas
* Python, PyTorch / TensorFlow
* Hugging Face Transformers
* LangChain / LlamaIndex
* FastAPI, React/HTML (Para el despliegue web)
